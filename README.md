# autolisp
Raccolta di funzioni utility in Autolisp puro per CAD AutoCAD e alternativi con base IntelliCAD 6, in particolare ProgeCAD 2009 Smart! che ha licenza gratuita per uso personale scaricabile qui: https://softradar.com/progecad-smart/

Collection of utility functions in Pure Autolisp for AUTOCAD and alternatives with IntelliCAD 6 base, in particular ProgeCAD 2009 Smart! who has a free license for personal use downloadable here: https://softradar.com/progecad-smart/
